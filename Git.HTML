Comandos Git Bash

Consola

ls -l En lista los archivos del directorio en el que nos encontremos.

clear limpia la pantalla.

cd Abrimos directorios. Ejemplo: cd ProyectoGit

cd .. Retrocedemos un directorios.

pwd Nos muestra la ruta del directorio.

code Git.html Abre el archivo de texto en visual studio code.

cat Git.html Nos deja visualizar el contenido del documento.

mkdir Crea una carpeta en el directorio. Ejemplo: mkdir ProyectoGit

touch Crea un documento en el directorio. Ejemplo: touch Git.html

history Nos muestra todos los comandos utilizados

Git

git init Inicia git.

git status Evidencia el estado de los documentos.
